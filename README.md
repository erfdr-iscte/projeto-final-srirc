# Projeto Final — SRIRC

Arquitetura resiliente baseada em microserviços críticos, com mitigação de ataques DoS e mecanismos de auto-recuperação.

## Tecnologias
- Docker
- Docker Compose
- Nginx
- Python (Flask)

## Estrutura
- api/ → microserviço crítico
- nginx/ → configuração do reverse proxy
- docker-compose.yml → orquestração dos serviços

## Objetivo
Demonstrar resiliência, balanceamento de carga e mitigação de ataques DoS num ambiente de microsserviços.
